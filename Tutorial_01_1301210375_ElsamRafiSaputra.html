  • Heading

    Heading pada HTML merupakan tag yang berguna untuk menampilkan judul dari konten laman web yang dibangun. Contoh sebagai berikut :

    Heading 1

    Heading 2

    Heading 3

    Heading 4

    Heading 5
    Heading 6

  • Hyperlink

    Hyperlink dalam HTML memungkinkan halaman web berpindah laman atau bernavigasi menuju laman web yang lain. Contoh sebagai berikut :

    Pendafataran Tiket VIP Dana Kaget 1 Triliun

  • Tabel

    Tabel pada HTML merupakan salah satu elemen penting khususnya digunakan untuk menampilkan data yang membutuhkan bentuk tabel. Contoh sebagai berikut :

    Nama Lengkap Kota Kelahiran Umur
    Budi Jakarta 35
    Andi Semarang 52
    Rasyid Surabaya 22

    Exercise dari PPT

    Nama Lengkap Gelar Pendidikan Age
    Sarjana Magister
    Budi S.Kom M.Sc 35
    Andi S.SiKom M.T 52

  • Image

    Image untuk menampilkan gambar pada halaman web merupakan sebuah improvisasi dalam pembuatan desain sebuah web yang dapat memperindah tampilan website.

  • Font Styling

    Contoh sebagai berikut :

    Ini font styling biasa

    Kalo ini font styling miring/italic

    Kalo ini ada garis dibawahnya/underline

    Kalo ini gendut font styling nya

    Kalo ini udah gendut, miring, digaris bawahi lagi

  • Line Break

    Sebagai seperti tulisan yang di enter atau dilewati satu line/baris

    test


    ini kelewat satu baris
